ROC / AUC Darkmode Cheatsheet

ROC & AUC：門檻掃過 + 排序能力

0) 主線一句話

ROC = 把所有 threshold 一次看完；AUC = 模型把正樣本排在負樣本前面的能力（排序能力）。

1) ROC 的兩個軸

Y 軸：TPR

TPR (True Positive Rate) = Recall / Sensitivity

X 軸：FPR

FPR (False Positive Rate) = 1 − Specificity

2) ROC 每一個點＝一個 threshold

  • threshold 低 → 判正較多 → TPR ↑、FPR ↑（點往右上走）
  • threshold 高 → 判正較少 → TPR ↓、FPR ↓（點往左下走）

3) 三個角落 + 對角線

左上角 (0,1)

TPR 高、FPR 低 → 近乎完美

右上角 (1,1)

threshold = 0 → 全部判正

左下角 (0,0)

threshold = 1 → 全部判負

對角線 y=x

隨機分類器：任何 threshold 都落在對角線上 → AUC = 0.5

4) AUC 的「機率」直覺

AUC = 隨機抽一個正樣本，它的分數比隨機抽一個負樣本更高的機率。

意思是：你隨便挑一對（正、負）來比，模型有多常把正的排在負的前面。

5) 用數字示範這個機率

正樣本分數：P1=0.9, P2=0.6
負樣本分數：N1=0.7, N2=0.2

四對比較：P1>N1 ✅、P1>N2 ✅、P2>N1 ❌、P2>N2 ✅

AUC = 3/4 = 0.75

6) 讀圖的快速規則（講義的重點）

Good vs Random

  • 曲線越靠左上越好
  • 貼近對角線＝接近亂猜

Threshold 方向

  • Threshold 下降 → 點往右上走
  • Threshold 上升 → 點往左下走

6.5) 看圖口訣：threshold 標籤怎麼讀

圖上的每個 t=0.1, 0.2, ... 就是一個門檻值：

  • 把分數（probability / score）轉成 0/1 的門檻：threshold = t
  • 規則通常是：score ≥ t → predict positive

直覺方向：

  • t 很大（例如 0.9）：很嚴格 → 很少人被判 positive → TPR 可能低、FPR 也低（靠左下）
  • t 很小（例如 0.1）：很寬鬆 → 很多人被判 positive → TPR 會上去、FPR 也會上去（往右上）

總結一句：

  • threshold ↓ → predicted positive ↑ → TPR ↑ 且 FPR ↑（往右上）
  • threshold ↑ → predicted positive ↓ → TPR ↓ 且 FPR ↓（往左下）

7) AUC 的「警覺」提示

  • AUC ≈ 0.5：隨機
  • AUC > 0.9：很強
  • AUC 接近 1：記得檢查資料洩漏（leakage）

8) ROC/AUC 與 Precision/Recall 的關係

概念你在看什麼重點
Precision / Recall單一 threshold 的表現要先選門檻
ROC / AUC所有 thresholds 的整體能力看排序能力

9) 實作提醒（Notebook 5b）

ROC 曲線是用 roc_curve 算出每個 threshold 的 (FPR, TPR)，AUC 用 roc_auc_score。

fpr, tpr, thresholds = roc_curve(y_true, y_prob) auc = roc_auc_score(y_true, y_prob)

每個 marker 就是一個 threshold；threshold 降低 → 走向右上。

10) Python 核心最短碼（可背）

1) 取分數（不是 0/1）

y_prob = model.predict_proba(X)[:, 1] # 正類分數

2) 畫 ROC + 算 AUC

from sklearn.metrics import roc_curve, roc_auc_score fpr, tpr, thresholds = roc_curve(y_true, y_prob) auc = roc_auc_score(y_true, y_prob)

3) 一個指定 threshold 的分類

t = 0.5 y_pred = (y_prob >= t).astype(int)

4) 你自己算 TPR / FPR（觀念對照）

TP = ((y_pred==1) & (y_true==1)).sum()
FP = ((y_pred==1) & (y_true==0)).sum()
TN = ((y_pred==0) & (y_true==0)).sum()
FN = ((y_pred==0) & (y_true==1)).sum()
TPR = TP / (TP + FN)
FPR = FP / (FP + TN)
